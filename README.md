# Data-analyst-Project-Dashboard-Supermarket-sales
##Objectives
-Analyze overall sales performance
-Identify top-selling products and categories
-Understand customer purchase patterns
-Track revenue trends over time
-Create clear and interactive reports using Excel
##Tools & Features Used
-Microsoft Excel
-Data Cleaning & Formatting
-Pivot Tables
-Pivot Charts
-Conditional Formatting
-Excel Formulas (SUM, AVERAGE, IF, COUNTIF, etc.)
-Data Validation
-Sorting & Filtering
##Key Insights Generated
-Total and average sales analysis
-Category-wise and product-wise sales performance
-High-revenue transactions identification
-Monthly/period-based sales trends
-Easy-to-read summary dashboards using charts
##Project Structure
-Raw Data Sheet – Original supermarket sales data
-Cleaned Data Sheet – Processed and formatted data
-Analysis Sheet – Pivot tables and calculations
-Dashboard / Charts – Visual representation of insights
##Learnings
-Hands-on experience with real-world sales data
-Improved data cleaning and preparation skills
-Practical understanding of Excel analytics
-Strong foundation for Power BI and SQL analysis
##KPIs for Dashboard
-Total Sales
-Total Orders
-Avg Order Value
-Total Quantity Sold
-Top Category
-Top Product
-Monthly Sales Growth
